# 2. Giới thiệu dự án

Dự án **Student Management Web App** là một ứng dụng web đơn giản dùng để quản lý thông tin sinh viên.

Người dùng có thể:

- Thêm sinh viên mới
- Xem danh sách sinh viên
- Xóa sinh viên
- Cập nhật thông tin sinh viên

Ứng dụng được xây dựng theo mô hình **Fullstack Web Application** gồm:

- Backend: cung cấp REST API
- Frontend: giao diện người dùng
- Database: lưu trữ dữ liệu sinh viên
